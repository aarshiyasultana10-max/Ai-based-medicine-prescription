## 🧠 AI-Based Medicine Prescription System

This project predicts diseases based on user symptoms and suggests suitable medicines, dosage, and precautions. It uses a dataset-driven chatbot interface with age-based filtering, order placement, and history tracking.

### 🚀 Features
- Disease prediction from symptoms  
- Medicine recommendation  
- Chatbot interface  
- Order tracking  

### 🛠️ Tech Stack
- HTML, CSS, JavaScript  
- Excel Dataset
